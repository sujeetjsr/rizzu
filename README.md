# Mega Offer Landing Page (Animated Version)
 
 ## How to Open
 Simply double-click on `index.html` file in your `rattre/ABPV` folder. It will open in your web browser.
 
 ## New Features
 - **Animated Header**: The background gradient now shifts colors.
 - **Scroll Reveal**: Images and elements will fade in and slide up as you scroll down.
 - **Confetti**: A burst of confetti appears when the page loads!
 - **Dynamic Title**: If you switch tabs, the browser tab title changes to "Wait! Don't Miss Out!" to grab attention.
 - **Urgency Timer**: The timer will shake when time is running low.
 
 ## Important: Setup WhatsApp Number
 Before sharing the page, you must add your WhatsApp Business number so the "Buy Now" button works.
 
 1. Open `script.js` in a text editor (like Notepad or VS Code).
 2. Look for the line near the bottom:
    ```javascript
    const whatsappNumber = 'INSERT_NUMBER_HERE';
    ```
 3. Replace `'INSERT_NUMBER_HERE'` with your actual number.
    - Example: `const whatsappNumber = '919876543210';`
    - **Do not** include `+` or spaces. just the country code and number.
 4. Save the file.
 
 ## How to Add/Remove Photos
 1. Put new photos in the `photo` folder.
 2. Open `script.js`.
 3. Add the new filename to the `images` list at the top of the file.
 
 ## Customizing
 - To change the price or text, edit `index.html`.
 - To change animation speeds or colors, edit `styles.css`.
